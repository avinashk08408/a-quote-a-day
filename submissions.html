Add Your Quote

Submit a quote and a short reflection.

1
Start

Sign in to GitHub

Open GitHub and sign in. Create a free account if needed.

2
Name

Name the file correctly

Use this format exactly:

2024-03-15-sunanda-vk.json
Date + your name · lowercase · no spaces
2024-03-16-arjun-r.json
Another example
File must end in .json
3
Paste

Fill the template

Copy the template and replace the values. 

{
  "quote":       "The quote you love.",
  "author":      "Quote author.",
  "contributor": "Your name.",
  "department":  "Your department.",
  "about":       "Why this quote matters to you."
}
"quote" Exact quote.
"author" Quote author.
"contributor" Your name.
"department" Your department.
"about" Why this quote matters to you.
Do not add an "id" field.
4
Submit

Create the Pull Request

Click:

1. “Propose new file” 2. “Create pull request”
An automated check will validate your file and comment on the PR.
